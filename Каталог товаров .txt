	Каталог товаров 
Функциональные требования:
 - Авторизация пользователей
 - Управление пользователями (CRUD)
 - Система ролей для пользователей
 - Добавление и редактирование товаров
 - Добавление отзывов на товар
 - Возможность создавать и редактировать заказы
 - Хранение информации о пользователях и их профилях
 - Журналирование действий пользователя

1. User 
 Хранит информацию о пользователях системы
 - idUser INT PK UQ NN AI - Уникальный идентификатор пользователя
 - login varchar (45) NN UQ - логин пользователя
 - password varchar(200) NN - пароль пользователя
 - idRole FK INT NN - связь с сущностью Role. Роль пользователя в системе
Связь один ко многим с сущностями Role,Order,Debit Card, Log.
Связь один к одному с сущностью Profile.

2. Role
Хранит роль пользователя
- idRole INT PK UQ NN AI - Уникальный идентификатор роли
- Role VARCHAR(30) NN - роль пользователя 
Планируется сделать 3 роли: 
ROLE_ADMIN - может просматривать информацию о пользователе, о заказах и магазинах, удалять товары если они не соответствуют описанию, удалять отзывы пользователей если те нарушают правила.
ROLE_USER - может оставлять отзывы на товар, оформлять заказы, изменять информацию о своем профиле, добавлять банковские карты для оплаты.
ROLE_OWNER - может добавлять новые товары в магазин, изменять их цену, изначально может зарегистрировать свой магазин и добавить промокоды.
Связь один к одному с сущностью User.

3. Profile
 Хранит дополнительную информацию о пользователе
 - idProfile INT NN AI PK UQ - уникальный идентификатор пользователя
 - username Varchar(30) NN - имя пользователя
 - address Varchar(80) NN - адрес для доставки
 - idUser FK INT NN - внешний ключ для связи один к одному с сущностью User
 - Phone number VARCHAR(12) NN UQ - телефонный номер пользователя
 - Email VARCHAR (100) NN UQ - электронная почта пользователя
Связь один к одному с сущностью User.
Связь один ко многим с сущностью Feedback.

4. Debit Card
 Хранит привязанные банковские карты пользователя
 - IdCard INT NN UQ PK AI - уникальный идентификатор для банковской карты 
 - Number varchar(16) NN UQ - номер банковской карты
 - idUser INT NN - внешний ключ для связи с сущностью User 
Связь один ко многим с сущностью User.

5. Log
 Хранит журнал активностей пользователя
 - IdLog INT NN AI UQ PK - уникальный идентификатор лога
 - IdUser INT NN - внешний ключ для связи с сущностью User
 - Action - VARCHAR (255) - активность которую выполнил пользователь.
 - Time TimeStamp(DEFAULT CURRENT_TIMESTAMP) - время выполнения этой активности
Связь один ко многим с сущностью User.

6. Order 
 Хранит информацию о заказе
 - idOrder INT NN PK AI UQ - уникальный идентификатор заказа
 - IdUser INT NN - внешний ключ для связи с сущностью User
 - Value DECIMAL (10,2) - хранит стоимость заказа
Связь один ко многим с сущностью User.
Связь многие ко многим с сущностью Product.

7. ProductsInOrder
 Хранит товары, которые находятся в заказе
 - IdProductsInOrder INT NN UQ AI PK -  уникальный идентификатор для товаров в заказе
 - idOrder INT NN - внешний ключ для связи с сущностью Order
 - idProduct INT NN - внешний ключ для связи с сущностью Product
 - Value INT NN - количество выбранного товара в заказе
Используется для связи многие ко многим

8. Product 
 Хранит информацию о товаре
 - IdProduct INT NN UQ AI PK -  уникальный идентификатор для товара
 - Name VARCHAR(100) NN - название товара
 - Description VARCHAR (300) NN - описание товара
 - Rate Decimal (1,2) NN - рейтинг товара
Связь многие ко многим с сущностью Order.
Связь один ко многим с сущностями Cost, Feedback.

9. Feedback
 Хранит информацию об отзыве на товар
 - IdFeedback INT NN UQ AI - уникальный идентификатор для отзыва на товар
 - idProfile INT NN - внешний ключ для связи с сущностью Profile
 - idProduct INT NN - внешний ключ для связи с сущностью Product
 - Description VARCHAR(300) NN
 - Rate INT NN
 Связь один ко многим с сущностями Profile, Product.

10. Price
 Хранит стоимость товара 
 - IdPrice INT AI NN UQ - уникальный идентификатор для цены товара
 - IdProduct INT NN - внешний ключ для связи с сущностью Product
 - IdShop INT NN - внешний ключ для связи с сущностью Shop
 - Value Decimal (10,2) - стоимость товара
Связь один ко многим с сущностями Product, Shop.

11. Shop
 Хранит информацию о магазине 
 - IdShop INT NN UQ AI PK - уникальный идентификатор для магазина
 - NAME VARCHAR (100) NN UQ - название магазина
 - Contacts VARCHAR (200) NN UQ - контакты магазина
Связь один ко многим с сущностями Coupon, Price.

12. Coupon
 Хранит информацию о купоне
 - IdCoupon INT NN AI UQ - уникальный идентификатор для купона
 - IdShop INT NN - внешний ключ для связи с сущностью Shop
 - Value INT NN - размер скидки по купону
 - Name VARCHAR (45) NN - название купона
Связь один ко многим с сущностью Shop.
